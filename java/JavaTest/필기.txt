자바 프로그램 실행 과정
1. "Hello.java" 작성
	- 사람이 사용하는 언어으로 구성
	- 컴퓨터는 이해 불가

2. 번역 > Compile(컴파일)
	- 번역 프로그램 > Compiler(컴파일러)
	- 자바 컴파일러 > javac.exe
	- javac.exe 파일명
	- javac.exe Hello.java
	- 산출물 > 번역된 결과본 > "Hello.class"

3. 실행
	- java.exe 파일명	
	- java.exe Hello
	- CPU가 명령어들을 차례로 실행
